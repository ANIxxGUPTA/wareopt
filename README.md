# WareOpt — Warehouse Labor & Delivery Slot Optimizer

WareOpt is a Warehouse Labor & Delivery Slot Optimizer.

## Structure
- `backend/` - Java 17, Spring Boot, Maven API
- `frontend/` - React, Vite, TypeScript UI
- `db/` - SQL migrations directory

## Running locally

Start the database:
```bash
docker-compose up -d
```
